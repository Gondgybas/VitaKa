# VitaKa — Учёт комплектующих на заводе

Программа для учёта комплектующих (винтов, болтов, гаек и т.д.) с автоматическим сохранением в Excel.

## Требования

- Python 3.8 или новее
- Операционная система: Windows, macOS или Linux

## Установка

1. Убедитесь, что установлен Python 3.8+:
   ```bash
   python --version
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск

```bash
python main.py
```

## Возможности

### Вкладка «Комплектующие»

- **Добавление** комплектующего с полями:
  - Тип (Винт, Болт, Гайка, Шайба и др.)
  - Диаметр (М3, М4, М6 и др.)
  - Длина (мм)
  - Количество
  - Вес на единицу (кг)
  - Произвольные дополнительные параметры (название + значение)
- **Редактирование** — двойной клик или кнопка «Редактировать»
- **Удаление** с подтверждением
- **Фильтрация** в стиле Excel — клик по заголовку столбца открывает меню с чекбоксами для выбора значений

### Вкладка «Журнал изменений»

- Автоматическая запись всех добавлений, изменений и удалений
- Каждая запись содержит: дату/время, операцию, название комплектующего, описание изменений
- Добавление пользовательского комментария к любой записи (двойной клик или кнопка)
- Сортировка по любому столбцу (клик по заголовку)

### Меню «Файл»

- **Настройки** — выбор папки для хранения Excel-файла
- **Открыть папку с Excel** — открыть проводник в папке с файлом данных
- **Выход**

## Хранение данных

Данные сохраняются автоматически после каждого изменения в файл Excel:

```
<выбранная папка>/vitaka_components.xlsx
```

Файл содержит два листа:
- **Комплектующие** — текущий список
- **Лог** — журнал всех изменений

Путь к папке сохраняется в `vitaka_settings.json` рядом с программой.
