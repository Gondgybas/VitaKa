# 🔩 VitaKa — Учёт комплектующих

**VitaKa** — это настольная программа для удобного учёта комплектующих на производстве. Она позволяет вести базу данных крепежа и других деталей, отслеживать все изменения и быстро находить нужные позиции с помощью гибких фильтров.

---

## 📋 Возможности

### 🗃️ Ведение базы комплектующих

- Добавление, редактирование и удаление записей о комплектующих
- Поддержка основных типов крепежа: винты, болты, гайки, шайбы, шпильки, заклёпки, штифты и другие
- Для каждой позиции можно указать:
  - **Тип** комплектующего
  - **Диаметр** (стандартные метрические размеры М2–М20 и другие)
  - **Длина** (в мм)
  - **Количество** на складе
  - **Вес** единицы (в кг)
- Возможность добавлять **произвольные дополнительные параметры** — любые характеристики, которых нет в стандартном наборе полей

### 🔍 Фильтрация в стиле Excel

- Нажмите на заголовок любого столбца в таблице — откроется меню фильтрации
- Выберите нужные значения с помощью чекбоксов
- Используйте поиск внутри фильтра для быстрого нахождения нужных значений
- Кнопки «Все» и «Сброс» для удобного управления выборкой
- Индикатор активных фильтров, чтобы вы всегда знали, что данные отфильтрованы
- Мгновенный сброс всех фильтров одной кнопкой

### 📖 Журнал изменений

- Каждое действие автоматически записывается в журнал:
  - Добавление нового комплектующего
  - Редактирование существующего (с указанием, что именно изменилось)
  - Удаление
- Журнал фиксирует **дату и время**, **тип операции**, **наименование** комплектующего и **подробное описание** изменений
- Возможность добавлять **комментарии** к записям журнала (например, причина изменения или примечание)
- Сортировка журнала по любому столбцу

### 💾 Хранение данных в Excel

- Все данные хранятся в формате **Excel (.xlsx)** — это удобно, понятно и легко переносится
- Файл содержит два листа:
  - **«Комплектующие»** — вся база деталей
  - **«Лог»** — полная история изменений
- Автоматическое сохранение после каждого действия — ваши данные не потеряются
- Файл Excel можно открыть в любой момент для просмотра или отправки коллегам

### ⚙️ Настройки

- Возможность выбрать **папку хранения** файла с базой данных
- При смене папки существующие данные автоматически переносятся
- Быстрый доступ к папке с данными прямо из меню программы
- Программа запоминает **размер и положение** окна между запусками

---

## 🚀 Как начать работу

1. **Запустите программу** — при первом запуске автоматически создастся файл базы данных
2. **Добавьте комплектующие** — нажмите кнопку «➕ Добавить» и заполните характеристики детали
3. **Управляйте записями** — редактируйте или удаляйте позиции по мере необходимости
4. **Используйте фильтры** — нажимайте на заголовки столбцов для быстрого поиска нужных деталей
5. **Следите за историей** — переключитесь на вкладку «Журнал изменений» для просмотра всех операций

---

## 🖥️ Интерфейс

Программа имеет интуитивно понятный интерфейс с двумя основными вкладками:

| Вкладка | Описание |
|---------|----------|
| **Комплектующие** | Основная таблица со всеми деталями, кнопки управления и фильтры |
| **Журнал изменений** | Полная история всех операций с возможностью добавления комментариев |

В нижней части окна расположена **строка состояния**, которая показывает время последнего сохранения и путь к файлу базы данных.

## 📦 Зависимости

- Python 3.10+
- `tkinter` — графический интерфейс (входит в стандартную поставку Python)
- `pandas` — работа с данными
- `openpyxl` — чтение и запись Excel-файлов
